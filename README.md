The Account Management System is a console-based Java application that simulates basic banking operations. The system allows users to create and manage bank accounts, perform transactions, and view account details securely.

This project demonstrates the implementation of Object-Oriented Programming (OOP) concepts such as encapsulation, abstraction, and class interaction using Java.

objective ------

To design a simple banking system using Java.

To implement real-world banking operations like credit, debit, and balance inquiry.

To generate unique account numbers and account IDs.

To practice exception handling and input validation.
